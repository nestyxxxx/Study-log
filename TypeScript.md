Типы переменных:
- number 
let temperature: number = 4
- string
- boolean
let isLoggedIn: boolean = true
- array (Массив значений одного типа) 
let cities: string[] = ["Moscow", "Berlin", "Tokyo"];
- enum (Перечисление — набор фиксированных значений)
enum Role {  
	Admin,  
	User,  
	Guest  
}  
let userRole: Role = Role.Admin;
- any (Любой тип. Отключает проверку типов)
- void (функция ничего не возвращает)

Typeof - проверяет примитивные типы данных
instanceof - показывает, является ли объект экземпляром определённого класса или конструктора

Типизация стандартных объектов JS:
- Map — объект со структурой данных в формате «ключ-значение». Один из наиболее удобных способов хранения данных и управления ими.(set, get)
- Set — коллекция уникальных значений, позволяет хранить только уникальные элементы.(add,has)
- Date — объект для работы с датой и временем.
- Promise — этот объект используется для асинхронных операций и обещает вернуть результат в будущем (либо успешный, либо ошибку).

Тип Awaited<Type>, который можно использовать для извлечения типа значения, возвращаемого из асинхронной функции с помощью оператора await.
